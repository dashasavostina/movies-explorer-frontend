# Фронтенд дипломного проекта «Movies Explorer»

## Стек технологий
- React
- NodeJS
- express.js

### `npm start`

Запускает приложение в режиме разработки.
Страница будет перезагружена после внесения изменений.


### `npm run build`

Запускает сборку приложения.
